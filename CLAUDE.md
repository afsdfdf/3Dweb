# Claude Code 项目配置

## 项目概述
这是一个 Payload CMS + Next.js 全栈项目，包含：
- CMS 内容管理
- AWS S3 媒体存储
- Stripe 支付集成
- AI 任务处理端点

## 审计重点
1. **安全性**：检查 API 端点、环境变量、权限控制
2. **性能**：数据库查询、API 响应时间、资源加载
3. **代码质量**：TypeScript 类型、错误处理、代码规范
4. **依赖管理**：过时包、安全漏洞
5. **配置完整性**：环境变量、部署配置

## 注意事项
- 不要修改 `.env` 文件中的敏感信息
- 数据库文件 `payload.db` 不要直接修改
- 所有修改需要保持向后兼容

## 权限设置
建议使用 `acceptEdits` 模式以便快速审计和修复
