# 2026 Archive

## Purpose

This folder stores dated, transitional, or process-heavy documents that are still worth keeping for historical reference but should not compete with the active engineering docs in `docs/`.

## What Belongs Here

- audit reports
- rollout checklists
- cutover plans
- migration memos
- worklogs
- temporary planning documents

## Rule

If an archived document still contains guidance that future work must follow, move that guidance into an evergreen document in the root `docs/` directory or into `docs/AI_PROJECT_MEMORY.md`.
